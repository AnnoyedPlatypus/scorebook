Noobs Cricboard is a free and open-source web application that aims to replace the pan and paper score counting tradition for a running cricket match. It offers users to count the score, wickets, run state to keep track of the match ball by ball.

This is single page application developed using vannila javascript, bootstrap5 for the user interface designing. The system is still under development.

Some of the features still need to be added. the code can be found at https://github.com/mahfuzurrahman98
